# Dossier .debug

Fichiers HTML et scripts de débogage pour le développement.

## Structure

- **html/** : Pages HTML de test et débogage
  - `check-*.html` : Vérification de données (localStorage, Supabase, CardMarket)
  - `clear-*.html` : Nettoyage de caches spécifiques
  - `debug-*.html` : Pages de debug pour composants spécifiques
  - `diagnose-*.html` : Diagnostic de problèmes
  - `test-*.html` : Tests de fonctionnalités

## Usage

Ouvrir ces fichiers directement dans le navigateur pour déboguer l'application.
